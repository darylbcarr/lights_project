Pairing Steps

  1. Find the discriminator in the display output:
    Disc: XXXX
  2. Enter the discriminator above and click Generate.
  3. If BLE fast advertising has timed out (>30 s since boot), type matter-pair at the clock> UART prompt before opening Alexa — the console is unresponsive once Alexa's PASE handshake begins.
  4. In Alexa → Add Device → Matter → Setup without QR code and enter the 11-digit manual code.
    Alexa validates QR codes against the Matter DCL; unregistered dev devices (VID=0) are silently rejected. The manual code bypasses this check.
  5. If re-pairing after a firmware update, erase the device NVS first:
    python -m parttool -p /dev/ttyUSB0 erase_partition --partition-name nvs
